## 📁 文件上传与配置

### 方法一：通过Web界面手动上传

1. **解压项目文件**
```bash
unzip brain-inspired-ai-complete.zip
cd brain-inspired-ai
```

2. **初始化Git仓库**
```bash
git init
git add .
git commit -m "Initial commit: Brain-inspired AI system foundation"
```

3. **添加远程仓库**
```bash
git remote add origin https://github.com/你的用户名/brain-inspired-ai.git
git branch -M main
git push -u origin main
```

### 方法二：通过GitHub Desktop (推荐)

1. **下载GitHub Desktop**
   - 访问: https://desktop.github.com/
   - 下载并安装GitHub Desktop

2. **克隆仓库到本地**
   - 在GitHub Desktop中选择 "Clone a repository from the Internet"
   - 选择你刚创建的仓库
   - 选择本地路径

3. **复制所有项目文件**
   - 将解压后的所有文件复制到本地仓库文件夹
   - GitHub Desktop会自动检测变更

4. **提交和推送**
   - 在GitHub Desktop中查看变更
   - 填写commit消息
   - 点击 "Commit to main"
   - 点击 "Push origin"
